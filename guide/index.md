## Quick Reference

| Task | Where to Go |
|------|-------------|
| Log mood, energy, activity | Tap the + button on the main screen |
| Add a note | Tap the note icon in the add modal |
| View past weeks | Swipe left/right on the main chart |
| See metrics grid | Swipe right from the main chart |
| Access settings | Menu → Settings |
| Set up habits | Menu → Extras → Habits |
| Backup your data | Menu → Settings → Backup |
